# minerU 模块技术总结

## 模块定位
`minerU/` 负责面向复杂文档的结构化提取与视觉增强解析，重点覆盖元数据、目录和图像区域理解。

## 目录概览
- `mineru_extractor.py`：主提取流程，组织 PDF 读取、视觉调用和元数据抽取。
- `parser/`：PDF 解析、图片解析、Office 转 PDF 等子流程。

## 核心能力
- 从对象存储中读取 PDF 或转换后的文档流。
- 调用视觉模型提取元数据字段与目录结构。
- 为图片、图表、目录页和文档首页提供增强理解。
- 支持时间字段标准化与结构化 JSON 输出。

## 与系统链路的关系
- `rag/svr/task_executor.py` 会调用 `minerU` 进行文档解析与字段抽取。
- `conf/magic-pdf.json` 和模型目录会影响其运行效果。

## 适用场景
- 扫描 PDF、报告、论文、册本类文档。
- 需要抽取标题、前言、正文、目录、发布日期等结构字段的场景。
