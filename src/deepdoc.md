# deepdoc 模块技术总结

## 模块定位
`deepdoc/` 负责深度文档理解，重点处理 OCR、版面识别、表结构识别以及复杂格式文档解析。

## 目录概览
- `vision/`：OCR、布局识别、TSR 测试与视觉模型调用。
- `parser/`：PDF、Office 文档及简历类结构化解析逻辑。

## 核心能力
- 从图片或 PDF 中提取文本。
- 识别标题、正文、图表、页眉页脚等版面结构。
- 对表格结构进行重建，并转换成适合 LLM 消化的自然语言表达。
- 为复杂文档提供更高精度的结构化输入。

## 工程价值
- 为 RAG 切片提供比纯文本提取更稳定的结构基础。
- 为论文、报告、扫描件、表格密集型文档提供更高质量的上游数据。

## 与其他模块的关系
- `rag/` 使用 `deepdoc/` 产出的结构化内容构建分块与索引。
- `driver/` 可参与模型准备。
- `minerU/` 与其形成互补的视觉文档解析能力。
